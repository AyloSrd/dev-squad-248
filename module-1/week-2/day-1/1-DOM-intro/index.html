D.O.M : Document Object Model

DOM is an API, designed to give you full control on any HTML/XML document.
It's an object-based system where any element on the page has a counter as a JS object.
Each kind of HTML Object has a predefined API (methods, poperties).
ALWAYS THINK OF THE DOCUMENT HAS A TREE !!!
