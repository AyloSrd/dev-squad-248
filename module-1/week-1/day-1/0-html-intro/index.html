Shell (terminal) Survival Guide

  • cd dir-name (change directory)
  • cd .. (go one level up in the file system, starting from the current position)
  • cd ./../../../../ (go four level up in the file system, starting from the current position)
  • ./ (represents the current position on teh fike system)
  • rm file-name (remove a file)
  • rm -rf folder-name (remove a file)
  • ls (list all files + folders)
  • ls -a (list all files + folders + hidden ones)
  • pwd (path working directory)
  • clear (empty the shell)
  • on your keyboard access previous/next commands with up/down arrows
  • mkdir folder-name (make folder
  • mv resource-location new-resource-location
  • nano file-name (open a basic text editor in the terminal)
  • vi | vim file-name (open a more customisable text editor in the terminal
